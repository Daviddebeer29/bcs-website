# BCS Website — Image & Icon Generation Prompts

> Use these prompts in **Midjourney**, **DALL·E 3**, **Adobe Firefly**, or **Stable Diffusion**.
> Each prompt is written for maximum quality and relevance to the BCS brand: dark industrial, electric blue accents, professional engineering aesthetic.

---

## How to Use

| Tool | Tips |
|------|------|
| **Midjourney** | Paste prompt after `/imagine`. Append `--ar 1:1 --style raw --v 6` for icons; `--ar 16:9 --v 6` for hero/section images |
| **DALL·E 3** | Paste directly. Add `"square format, no text"` for icons |
| **Adobe Firefly** | Use the prompt as-is. Set style to "Photo" for photos, "Graphic" for icons |
| **Stable Diffusion** | Use with `DPM++ 2M Karras` sampler, 30–40 steps, CFG 7 |

**Brand colours to reference in tools that support colour palettes:**
- Primary dark: `#07070D`
- Electric blue: `#2A7FD4`
- Cyan accent: `#00D4FF`
