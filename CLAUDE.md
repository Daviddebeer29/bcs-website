# BCS Website - Barnard Control Systems

## Project
Static landing page for Barnard Control Systems, an electrical contractor in Western Cape, South Africa.

## Stack
- Pure HTML/CSS/JS (no framework)
- Deployed on Vercel as static site
- Single index.html entry point

## Design System
- Theme: "Steel Authority" - dark navy (#0a0f1c), cyan accents (#00d4ff)
- Typography: Industrial/technical feel
- Layout: Full-width sections, scroll-based single page

## Known Issues to Fix

## Company Info
